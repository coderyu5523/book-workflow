# 글로벌 코드 규칙

모든 에이전트가 준수하는 코드 표시 규칙.

---

## 코드 블록

- 모든 코드 블록. 언어 태그 필수 (`java`, `python`, `bash` 등)
- 구조. 목적 한 줄 → 코드 → 설명 → 실행 결과
- 핵심만 10~15줄 (전체는 versions/ 참조)
- bash/shell. 상세 설명 불필요
- 번호 주석(①②③) 금지

## 파트별 코드 규칙

- 이야기 파트에 코드 블록 절대 없음
- 기술 파트에서만 코드 등장
- 원본 코드가 진실의 원천 (불일치 시 챕터를 코드에 맞춤)

## 코드 파일 유형

- [실습] 파일. 전체 코드. "아래 코드를 `파일명`에 작성합니다"로 시작
- [설명] 파일. 핵심 흐름만 발췌. 전체 코드는 `code/exNN/` 참조
- [참고] 파일. 코드 블록 없음. 메서드 표로 요약
